# Advant Test Task

## Описание проекта
Это тестовое задание для компании Advant, реализованное с использованием Unity и архитектуры ECS (Entity Component System).

## Ссылка на проект
[Играть онлайн](https://play.unity.com/en/games/3c72fcc9-640f-4d72-9076-daa5e1ee4686/advant-test-task)

## О проекте
Проект представляет собой бизнес-симулятор, где игрок может управлять различными бизнесами. Основные возможности:
- Покупка и улучшение бизнесов
- Система уровней для каждого бизнеса
- Два типа улучшений для каждого бизнеса
- Автоматический доход от бизнесов
- Визуальное отображение прогресса дохода

## Техническая реализация
Проект реализован с использованием архитектуры ECS (Entity Component System), что обеспечивает:
- Четкое разделение данных и логики
- Высокую производительность
- Масштабируемость кода
- Легкость поддержки и расширения

### Основные компоненты:
- `Business` - компонент, содержащий данные о бизнесе
- `IncomeProgress` - компонент для отслеживания прогресса дохода
- `LevelUpRequest` - компонент для запроса повышения уровня
- `UpgradeRequest` - компонент для запроса улучшения бизнеса

### Системы:
- Система обработки дохода
- Система улучшения бизнесов
- Система повышения уровней

## Использованные технологии
- Unity
- ECS (Entity Component System)
- C#
- UI Toolkit
